# Ferramentas Católicas Online

Hub de ferramentas gratuitas para facilitar o dia a dia pastoral da Igreja Católica no Brasil.

## Stack

- Next.js 14 (static export)
- TypeScript
- Tailwind CSS
- Lucide React

## Desenvolvimento local

```bash
pnpm install
pnpm dev
```

Acesse: http://localhost:3000

## Estrutura

- `apps/web`: frontend
- `workers`: Cloudflare Workers
- `packages/shared`: tipos e constantes compartilhados
